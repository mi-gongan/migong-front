# migongApp( 마이그레이션 및 리팩토링 )

### 원래 사이트

https://migong.vercel.app/

### 추가 설명 사이트

https://migong.oopy.io/

#### 해야할 것

1. CSR->SSR(NEXT.js)
2. 상태관리 redux 사용
3. 리팩토링
4. 랜더링 최적화
5. 재사용성를 위한 컴포넌트간 종속성을 없애는 방향으로 분리하고 폴더 구조 개선

## 환경 설정

```
npx create-next-app@latest --typescript
```
